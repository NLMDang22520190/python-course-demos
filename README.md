# Python Course Demos

This repository contains two educational Python projects designed for beginners and intermediate learners.

## Projects Overview

### Project 1: `01_Python_Start_Turtle` (Spirograph)

A visually appealing geometric drawing script using Python's built-in `turtle` module. This project demonstrates loops, iteration, and basic coordinate geometry without needing any external libraries.

### Project 2: `02_Python_Pro_Pygame` (Catch the Apple)

A classic, simple 2D arcade game built with `pygame`. The player controls a basket to catch falling apples. It demonstrates core game development concepts including the game loop, event handling, and collision detection.

---

## Prerequisites

- Python 3.x installed on your system.

## Setup Instructions

1. Clone or download this repository.
2. Open a terminal and navigate to the `python-course-demos` folder.
3. Create a virtual environment to keep dependencies isolated:
   ```bash
   python3 -m venv .venv
   ```
4. Activate the virtual environment:
   - **macOS/Linux**: `source .venv/bin/activate`
   - **Windows**: `.venv\Scripts\activate`
5. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## How to Run

### Run Project 1: Spirograph (Turtle)

```bash
python 01_Python_Start_Turtle/spirograph.py
```

### Run Project 2: Catch the Apple (Pygame)

Ensure your virtual environment is active, then run:

```bash
python 02_Python_Pro_Pygame/catch_the_apple.py
```
